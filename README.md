# 비트코인 자동 매매 프로그램 (미완)
- 목표
  - 업비트 api를 이용한 비트코인 매매 프로그램 구현
  - 주요 코인 시계열 분석 예정

- files
  - request_order.py : 주문, 주문 확인 함수 구현
  - request_chart.py : 시세캔들 정보 호출 함수 구현
  - R_ts_functions.py : R에서의 시계열 함수 구현
  - ts_analysis.py : 거래소 차트 기능 함수 구현 (ex. 이동평균선)
  - trader.ipynd : 매매로직 구현

- 예정
  - 시계열 분석
  - 매수 매도 조건 구상
  - 자동화 프로그램
  - 거래시 메시지 전송
